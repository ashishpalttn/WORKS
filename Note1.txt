Weather Dashboard App
Description:
Build a Weather Dashboard where users can:

Search for a city to view its current weather.
See details like temperature, wind speed, and weather conditions.
View and select from a history of searched cities.

Requirements:
Use the Open-Meteo Geocoding API to get latitude and longitude for a city.
Use the Open-Meteo Weather API to fetch current weather data.
Manage state using Redux for:
Current weather data.
Search history.

Bonus:
Handle invalid city names or API errors gracefully.
Add a loading indicator during API calls.
Persist search history in local Storage.
